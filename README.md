# DrawingApp
A simple drawing canvas app to take quick notes or for children to draw.

## Libs and Techniques

* [Kotlin](https://kotlinlang.org/)
* [Coolors](https://coolors.co) 
* [Android Studio](https://developer.android.com/docs)
